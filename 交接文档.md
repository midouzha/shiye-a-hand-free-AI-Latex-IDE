# 师爷LaTeX排版工具 交接文档

## 1. 当前可交付范围

本仓库已完成可运行的最小闭环：
- Step2 问答式交互（界面逐题采集）
- Step3 大模型内容生成（通过 `call_example.py` 配置客户端）
- Step4 LaTeX 渲染、编译与自动修复
- Step5 PyQt 主界面、PDF 预览、生成历史列表

## 2. 运行环境

已提供环境文件：
- `environment.yml`
- `requirements.txt`

建议使用专用环境：
- `conda activate shiye-latex`

## 3. 启动方式

在项目根目录执行：

```bash
python scripts/run_step5_app.py
```

## 4. 关键目录说明

- `app/workflow/`：问答与流程编排
- `app/llm/`：大模型请求客户端与内容生成
- `app/latex/`：模板渲染、编译、错误解析与自动修复
- `app/ui/`：主界面与生成控制器
- `scripts/run_step5_app.py`：主入口

## 5. 提交前检查建议

1. 在 `shiye-latex` 环境下启动 UI，确认可正常打开。
2. 走一轮问答并点击生成，确认 PDF 能产出并可预览。
3. 检查 `call_example.py` 中密钥管理策略是否符合仓库安全要求。
